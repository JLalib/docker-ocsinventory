## docker-compose de OCS Inventory
### Instalación
Primer paso <br><p>
Modifica o copia el fichero php.ini <br><p>
Crea o copia el fichero docker-compose.yml e inicia el contenedor. <br><p>
### Acceso vía web
http://localhost:8080/ocsreports<p>
